# Get-ExternalIP
 
The module returns the external IP address of the requesting computer.
<!--more-->

 It performs a call to http://ifconfig.me/ip and returns data.
 
 <!--og-image-->
 ![Diagram](Images/Diagram.png)

 Usage:

 ```Powershell
 Get-ExternalIP
 ```
